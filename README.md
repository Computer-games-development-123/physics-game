# פרויקט משחק בסגנון Icy Tower עם רכיבים פיזיקליים

פרויקט זה פותח כחלק ממטלת קורס בפיתוח משחקי Unity. המשחק משלב תכנון
אדריכלי (Architecture), עקרונות OOP, פיזיקה, אפקטורים, ניהול מחלקות, UI
ודוחות.\
המשחק נבנה בהשראת **Icy Tower** וכולל מגדל אינסופי, קפיצות מדויקות,
פלטפורמות מגוונות, אזורי רוח, אזור מוות, וניקוד דינמי.

------------------------------------------------------------------------

## ✨ תיאור כללי של המשחק

המשחק הוא משחק פלטפורמה אנכי שבו השחקן מטפס במעלה מגדל אינסופי על ידי
קפיצה מפלטפורמה לפלטפורמה.\
ככל שהשחקן עולה גבוה יותר -- הניקוד עולה.\
נפילה מתחת לגבול התחתון של המסך מובילה למסך Game Over.

המשחק כולל: - פלטפורמות רגילות - פלטפורמות קפיץ (Spring) - אזור רוח
(AreaEffector2D) - פלטפורמות חלקות (Ice Platform) עם Physics Material -
מצלמה שעוקבת רק כלפי מעלה - מערכת ניקוד חיה - מסך Game Over עם כפתור
Reload - מגדל אינסופי שנוצר פרוצדורלית

# 🧩 רכיבי ליבה במשחק

## 🎮 PlayerMovement.cs

מטפל בתנועה: - תנועה אופקית באמצעות Input - קפיצה - בדיקה אם השחקן על
הקרקע - Rigidbody2D לניהול פיזיקה - מניעת קפיצה כפולה

## 🧰 PlatformSpawner.cs

אחראי ליצירה דינמית של פלטפורמות: - יוצר פלטפורמות רגילות - לעיתים מייצר
SpringPlatform לפי הסתברות - מוודא שלא יהיו שתי פלטפורמות קפיציות ברצף
(10--20 פלטפורמות רגילות ביניהן) - מוודא תמיד שיש מגדל מעל השחקן

## 🌀 SpringPlatform.cs

פלטפורמה מיוחדת שמקפיצה את השחקן: - PhysicsMaterial2D עם Bounciness -
Boost בקוד כדי להבטיח קפיצה חזקה ועקבית - זיהוי מגע מלמעלה בלבד

## 🌬️ Wind Zone

אזור רוח באמצעות: - AreaEffector2D - BoxCollider2D כ־Trigger - מאפשר
"דחיפה" אנכית (או אופקית)

## ❄️ IcePlatform

מבוסס על PhysicsMaterial2D: - חיכוך נמוך מאוד (החלקה) - משמש כרכיב
פיזיקלי נוסף

## ☠️ DeathZone.cs

אובייקט שממוקם מתחת למצלמה: - מזהה נפילה של השחקן - מציג מסך Game Over -
עוצר את זמן המשחק - מאפשר Restart

## 🧮 ScoreManager.cs

מחשב את הניקוד: - מבוסס על גובה מקסימלי שהשחקן הגיע אליו - מציג Score חי
במהלך המשחק - מעביר את הניקוד למסך Game Over

# 🏗️ אפקטורים ומנוע פיזיקלי

המשחק מנצל מגוון אפקטורים:

### ✔ Platform Effector 2D

משמש לפלטפורמות חד־כיווניות --- מאפשר מעבר מלמטה ונחיתה מלמעלה.

### ✔ Area Effector 2D

יוצר אזורי רוח המשפיעים על Rigidbody2D.

### ✔ Physics Material 2D

משמש ליצור: - SpringMaterial (קפיציות גבוהה) - IceMaterial (חיכוך כמעט
0)

### ✔ Rigidbody2D + Colliders

משמשים לכל אובייקטי הפיזיקה במשחק.

# 🧠 עקרונות OOP ואדריכלות בקוד

### ✔ Single Responsibility Principle

כל סקריפט אחראי לפונקציה אחת: - תנועה - ניקוד - יצירת פלטפורמות - קפיץ -
אזור מוות

### ✔ Composition over Inheritance

רוב ההתנהגות מבוססת על קומפוננטים ולא על ירושה.

### ✔ Separation Between Logic & Presentation

-   פלטפורמות = Logic
-   UI = Presentation
